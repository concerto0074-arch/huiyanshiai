随机森林算法

分类算法

乳腺癌 肺癌
算法原理

随机森林是一种集成学习算法，通过构建多个决策树并将它们的结果进行组合来提高预测准确性。它的核心思想是"三个臭皮匠，顶个诸葛亮"，通过集成多个弱分类器（决策树）来构建一个强分类器。

随机森林的工作原理包括两个关键步骤：

  • 随机抽样（Bootstrap）：从原始数据集中有放回地随机抽取样本，构建多个训练数据集。
  • 随机特征选择：在每个决策树的每个节点，从所有特征中随机选择一部分特征，然后从中选择最佳分割特征。

最终的预测结果通过投票（分类问题）或平均（回归问题）的方式由所有决策树共同决定。

  • 准确率高、抗过拟合
  • 能处理高维数据
  • 可评估特征重要性
  • 适合大规模数据集
  • 对缺失值不敏感
  • 能处理类别不平衡数据
算法类型: 分类算法
适用数据: 结构化数据

适用场景
  • 结构化数据分类：如医学诊断、金融风险评估等。
  • 特征重要性分析：帮助理解哪些特征对预测结果影响最大。
  • 大规模数据集：由于其并行化特性，适合处理大规模数据。
  • 类别不平衡数据：对不平衡数据有较好的处理能力。

支持向量机(SVM)算法

分类算法

乳腺癌 肺癌
算法原理

支持向量机（SVM）是一种强大的分类算法，其核心思想是在高维空间中寻找一个最优超平面，将不同类别的样本分隔开，并且使两类样本到超平面的距离最大化。这个距离被称为间隔（Margin），SVM的目标就是找到具有最大间隔的超平面。

SVM的工作原理包括以下关键概念：

  • 超平面：在N维空间中，超平面是N-1维的子空间，用于分隔不同类别的样本。
  • 支持向量：距离超平面最近的样本点，它们决定了超平面的位置和方向。
  • 核函数：用于将低维非线性数据映射到高维线性可分空间，常见的核函数包括线性核、多项式核、高斯核（RBF）等。

对于非线性问题，SVM通过核函数将数据映射到高维空间，然后在高维空间中寻找线性超平面进行分类。

  • 适合处理高维数据
  • 对小样本数据效果好
  • 理论基础坚实
  • 泛化能力强
  • 可以通过核函数处理非线性问题
  • 对异常值不敏感
算法类型: 分类算法
适用数据: 结构化数据
关键参数
参数名称 默认值 说明
C 1.0 正则化参数，控制模型的复杂度和对误分类的惩罚程度。C值越大，模型越容易过拟合。
kernel rbf 核函数类型，可选值包括linear（线性核）、poly（多项式核）、rbf（高斯核）等。
gamma scale 核函数系数，仅用于rbf、poly和sigmoid核。gamma值越大，模型越容易过拟合。
degree 3 多项式核的次数，仅用于poly核。
coef0 0.0 核函数中的独立项，仅用于poly和sigmoid核。
适用场景
  • 高维数据分类：如文本分类、图像识别等。
  • 小样本学习：当训练数据量较小时，SVM通常能取得较好的效果。
  • 非线性分类问题：通过核函数可以处理复杂的非线性分类问题。
  • 异常检测：可以用于异常值检测和离群点识别。

  • 对于噪声较大的数据，SVM的性能可能不如其他算法（如随机森林）。

决策树算法

分类算法

乳腺癌 肺癌
算法原理

决策树是一种基于树形结构的分类算法，它通过递归地分割特征空间来构建预测模型。决策树的基本思想是将数据集逐步划分为更小的子集，同时在每个节点上选择最优的特征进行分割，直到所有子集都属于同一类别或达到预设的停止条件。

决策树的构建过程包括以下关键步骤：

  • 选择最优特征：根据信息增益、信息增益比或基尼指数等指标，选择能够最大程度提高节点纯度的特征。
  • 分割数据集：根据选定的特征，将数据集分割为多个子集。
  • 递归构建子树：对每个子集重复上述过程，直到满足停止条件。
  • 剪枝：通过剪枝操作减少树的复杂度，防止过拟合。

决策树的预测过程非常简单，从根节点开始，根据输入样本的特征值，按照树的分支规则向下遍历，直到到达叶节点，叶节点的类别即为预测结果。

  • 可解释性强
  • 训练速度快
  • 能处理多种类型特征
  • 无需特征归一化
  • 直观易懂，易于可视化
  • 能够处理非线性关系
算法类型: 分类算法
适用数据: 结构化数据
关键参数
参数名称 默认值 说明
criterion gini 特征选择标准，可选值包括gini（基尼指数）和entropy（信息增益）。
max_depth None 树的最大深度，控制树的复杂度，防止过拟合。
min_samples_split 2 节点分裂所需的最小样本数。
min_samples_leaf 1 叶子节点所需的最小样本数。
max_features None 每个节点分裂时考虑的最大特征数。
适用场景
  • 可解释性要求高的场景：如医疗诊断、金融风险评估等，需要清晰的决策过程。
  • 快速原型开发：训练速度快，适合快速构建模型和迭代。
  • 特征重要性分析：可以直观地了解哪些特征对预测结果影响最大。
  • 小规模数据集：在数据量较小时，决策树仍能取得较好的效果。

  • 对连续特征的处理相对复杂，可能需要离散化处理。
  • 在处理具有复杂非线性关系的数据时，性能可能不如其他算法（如SVM、神经网络）。

K近邻(KNN)算法

分类算法

乳腺癌 肺癌
算法原理

K最近邻（KNN）是一种基于实例的分类算法，其核心思想是"近朱者赤，近墨者黑"。KNN算法的基本原理是：对于一个新的样本，根据它最近的K个训练样本的类别来预测其类别。

KNN算法的工作过程包括以下关键步骤：

  • 计算距离：计算新样本与所有训练样本之间的距离（通常使用欧氏距离、曼哈顿距离或闵可夫斯基距离）。
  • 选择最近邻：根据计算得到的距离，选择与新样本最近的K个训练样本。
  • 投票决策：根据K个最近邻样本的类别，通过多数投票的方式确定新样本的类别。

KNN算法是一种惰性学习算法，它不需要训练过程，而是在预测时直接使用训练数据进行计算。

  • 简单易懂
  • 能处理非线性数据
  • 适合增量学习
  • 无需训练过程
  • 对异常值不敏感
  • 能处理多分类问题
算法类型: 分类算法
适用数据: 结构化数据
关键参数
参数名称 默认值 说明
n_neighbors 5 邻居的数量K，K值越小，模型越容易过拟合；K值越大，模型越稳定，但可能导致类别边界模糊。
weights uniform 投票权重，可选值包括uniform（均匀权重）和distance（距离权重，距离越近权重越大）。
algorithm auto 计算最近邻的算法，可选值包括auto、ball_tree、kd_tree和brute。
leaf_size 30 BallTree或KDTree算法的叶子节点大小，影响树的构建和查询速度。
p 2 距离度量的参数，p=1表示曼哈顿距离，p=2表示欧氏距离。
适用场景
  • 小规模数据集：在数据量较小时，KNN能取得较好的效果。
  • 增量学习场景：新样本可以直接添加到训练集中，无需重新训练模型。
  • 多分类问题：能自然地处理多分类问题，无需特殊处理。
  • 非线性分类问题：能处理复杂的非线性分类边界。

  • 对高维数据效果较差，存在维度灾难问题。
  • 对不平衡数据敏感，多数类样本可能主导预测结果。

算法使用指南

1. 选择算法

从上方的算法卡片中选择适合您需求的算法。每个算法卡片显示了算法的基本信息、性能指标和适用场景。

您可以根据以下因素选择算法：

  • 算法的准确率和召回率
  • 算法的适用数据类型
  • 算法的计算效率
  • 算法的可解释性
2. 填写病人数据

点击算法卡片下方的"使用模型"按钮，在弹出的模态框中填写病人的基本信息和医疗数据。

需要填写的信息包括：

  • 基本信息：姓名、年龄、性别
  • 生活习惯：吸烟史、BMI指数
  • 医疗数据：肿瘤标志物、基因突变情况
3. 查看预测结果

填写完数据后，点击"提交数据"按钮，系统将使用选择的算法进行预测。

预测结果将在新的模态框中显示，包括：

  • 风险等级：低风险、中风险或高风险
  • 置信度：预测结果的可信度
  • 风险分布：不同风险等级的概率
  • 建议：针对预测结果的健康建议
4. 下载报告

在预测结果模态框中，您可以点击"下载报告"按钮，将预测结果保存为HTML文件。

报告包含：

  • 完整的预测结果
  • 风险等级和置信度
  • 风险分布图表
  • 个性化健康建议
