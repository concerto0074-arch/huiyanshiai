# 程序说明文档 (R 脚本说明)

本项目包含一系列用于生物信息学分析的 R 脚本，主要针对 NSCLC（非小细胞肺癌）基因表达数据进行 WGCNA（加权基因共表达网络分析）和机器学习分群。

## 主要变更

1.  **编码修复**：所有 `.R` 脚本已统一转换为 **UTF-8** 编码，解决了在不同操作系统（如 Mac 和 Windows）下中文注释乱码的问题。
2.  **路径修复**：
    *   移除了所有硬编码的绝对工作路径（如 `C:/Users/Meng/...` 或 `C:/Users/acer/...`）。
    *   将 `setwd()` 调用全部注释掉。
    *   将所有数据读取 (`read.csv`) 和保存 (`write.csv`) 的路径更改为**相对路径**，确保代码在任何环境下均可运行。
3.  **脚本相互引用与数据流**：
    *   修正了部分脚本指向错误文件夹的问题（如将误用的 `新建文件夹` 修正为 `互信息`）。
    *   确保了数据生产脚本（如 `SU.R` 产生 `paixusu.csv`）与消费脚本（如 `proposed.R`）之间的路径一致性。

## 目录结构

*   `WGCNA干净数据/`：存放去噪后的干净数据及其分析脚本。
    *   `proposed.R`：核心算法脚本。
    *   `AMRSOGL.R`：另一种核心算法脚本。
    *   `xiaorong.R`：消融实验脚本。
*   `WGCNA原始数据/`：存放原始带噪数据及其分析脚本。
*   `AUC/`：存放模型表现评估（AUC）相关的脚本和结果。
*   `不重叠分群/`：存放基础 WGCNA 分群脚本。

## 运行建议

1.  运行脚本前，请确保 R 的当前工作目录（Working Directory）设置为该脚本所在的文件夹。
2.  部分脚本依赖于其他脚本生成的数据文件。例如，在运行 `WGCNA干净数据/proposed.R` 之前，建议先确认同目录下的 `SU.R`（生成权重文件）已运行或相关数据已存在。
3.  确保已安装必要的 R 包：`WGCNA`, `msgl`, `devtools`, `doParallel` 等。
